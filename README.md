# lol-inted-bot

A Discord bot that looks up your recent League of Legends games and uses AI to decide if **you** inted or if your **team** ran it down harder.

---

## Command: `?inted`

### Usage

In any text channel where the bot is present, type:


?inted Summoner Name#Tag

Example:
?inted CoolGamer123#NA1
?inted ProPlayer#1234

Shows a List of 5 recent matches for the given summoner. You can then pick a match to analyze.
